Provides the complete set of BSON {@link com.allanbank.mongodb.bson.Element}s used to construct {@link com.allanbank.mongodb.bson.Document}s.

Usage

Elements are normally constructed using a builder from the com.allanbank.mongodb.bson.builder package. The {@link com.allanbank.mongodb.bson.Visitor} interface provides a convenient method to traversing a Document without directly accessing these classes. The {@link com.allanbank.mongodb.bson.NumericElement} interface provides an easy way to coerce a numeric element value into an expected type.
